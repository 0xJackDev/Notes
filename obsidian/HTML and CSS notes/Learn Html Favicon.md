

This is how you can create a favicon for your webpage. This is the thing in the topleft of a tab. 


First find a image,  a .ico, .png or .jpg or more. 


put the image in your index.html folder


all you do is inside of the head attribute add a link tag with will add a relationship between the file and the link and what it does so type

	<link rel="icon" type="image/jpg" href="favicon.jpg"
replace favicon .jpg with the name of your image 


&copy adds the copyright symbol
